#PnP JavaScript Core#

### Summary ###
This sample is a consolidated set of JavaScript examples for use in your SharePoint/Patterns and Practices projects.

### Applies to ###
-  Office 365 Multi Tenant (MT)
-  Office 365 Dedicated (D)
-  SharePoint 2013 on-premises

### Prerequisites ###
